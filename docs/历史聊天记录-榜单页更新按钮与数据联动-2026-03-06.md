## 1. 需求：前端按钮触发数据更新

**用户需求**  
- 在前端页面增加按钮，点击后发起数据更新。  
- **Solana Pump 榜单**：按现有逻辑查询符合条件的代币并写入数据库。  
- **zhilabs 精选**：更新现有代币数据（从 ca.md 拉取 AVE 并 upsert 到 zhilabs_ranking）。

**实现**  
- **后端**（`src/server.js`）：  
  - 新增 `POST /api/update?type=pump|zhilabs`。  
  - `type=pump` 调用 `updatePumpRanking()`（来自 `scripts/fetch-pump-ranking.js`）。  
  - `type=zhilabs` 调用 `updateZhilabsRanking()`（来自 `scripts/fetch-zhilabs-ranking.js`）。  
  - 使用内存锁避免同一类型并发执行，返回 `{ ok, type, updated, durationMs }`。  
- **脚本可复用**：  
  - `scripts/fetch-pump-ranking.js` 导出 `updatePumpRanking`，仍支持 `npm run pump-ranking` 独立运行。  
  - `scripts/fetch-zhilabs-ranking.js` 导出 `updateZhilabsRanking`，仍支持 `npm run zhilabs-ranking` 独立运行。  
- **榜单页**（内联在 `src/server.js` 的 `HTML_PAGE`）：  
  - 在 Tab（Solana Pump 榜单 / zhilabs精选）下方增加一行：「刷新数据：」+ 按钮「更新 Pump 榜单」或「更新 zhilabs 精选」（随当前 Tab 切换文案）。  
  - 点击后 POST `/api/update?type=当前Tab`，展示「更新中…/完成/失败」及条数、用时，并刷新当前 Tab 表格。

## 2. 用户反馈：页面上看不到更新按钮

**用户反馈**  
- 前端页面没看到更新按钮。

**原因与处理**  
- 榜单页 HTML 由服务启动时读入内存，若未重启服务会一直返回旧页面。  
- 将「更新」操作单独成行（`.action-row`），放在 Tab 与说明文字之间，避免与 Tab 挤在一行被忽略。  
- 为更新按钮使用蓝色主按钮样式（`#1e40af`），与灰色 Tab 区分。  
- 增加「刷新数据：」标签 + 明确说明：更新按钮在 **`/ranking` 榜单页**，需先点首页「discovery」进入或直接访问 `http://localhost:3000/ranking`，并**重启服务后强制刷新**浏览器才能看到最新布局。

---

## 3. 需求：数据库更新后前端要联动

**用户需求**  
- 数据库更新了，前端页面没更新，需要联动。

**实现**  
- **禁止接口缓存**：为 `/api/ranking`、`/api/ranking/zhilabs`、`/api/update` 响应增加 `Cache-Control: no-store`，避免浏览器或中间层返回旧数据。  
- **切换 Tab 时刷新**：切换「Solana Pump 榜单」/「zhilabs精选」时，请求对应 API 并重绘当前 Tab 表格，并更新「最后同步」时间。  
- **展示最后同步时间**：在操作行增加「最后同步：HH:MM:SS」，在首次加载、点击「更新…」、切换 Tab 后更新。  
- 曾增加每 15 秒轮询刷新，见下节关闭。

---

## 4. 用户要求：不开启自动刷新以节省 AVE API 额度

**用户要求**  
- 先不要开启自动刷新，会对 AVE API 额度造成影响。

**实现**  
- 移除榜单页的定时轮询（原 15 秒一次刷新当前 Tab）。  
- 移除「自动刷新：开启（15s）」的展示与相关 `setSyncStatus` 逻辑。  
- 保留「最后同步」：仅在**用户点击「更新…」**或**切换 Tab** 时更新；这两类操作只读 Supabase 或用户主动触发 AVE，不增加自动请求。

## 6. 使用说明

- **榜单页**：访问 `/ranking`，在 Tab 下可见「刷新数据：[更新 Pump 榜单] 或 [更新 zhilabs 精选]」及「最后同步」时间。  
- **更新数据**：点击按钮会调用 AVE 并写库，完成后自动刷新当前 Tab 表格。  
- **仅看最新库数据**：切换 Tab 会重新请求对应 API 并刷新表格（不调 AVE）。  
- 修改榜单页或接口后需**重启服务**并**强制刷新**浏览器才能看到最新界面。
