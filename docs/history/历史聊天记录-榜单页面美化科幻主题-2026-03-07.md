# 历史聊天记录：榜单页面美化（科幻主题统一）

**日期**：2026-03-07  
**文件名**：历史聊天记录-榜单页面美化科幻主题-2026-03-07.md

---

## 1. 需求

**用户需求**  
- 美化榜单页面（`/ranking`），使其视觉风格与欢迎页（`src/public/index.html`）的深空科幻主题保持一致。

**美化前状态**  
- 榜单页使用 `system-ui` 字体，`#0f0f12` 纯色背景，基础 `<table>` 表格样式。
- Tab 按钮为简单的圆角矩形，无动画效果。
- 排名仅为纯数字，无视觉区分。
- 与欢迎页的 Orbitron/Exo 2 字体、星空粒子、星云渐变、品牌 Logo 等视觉元素脱节严重。

---

## 2. 实现方案

### 2.1 背景层

在 `<body>` 中添加四层固定背景（`position: fixed`），模拟欢迎页的深空氛围：

| 层级 | 类名 | 效果 |
|------|------|------|
| 1 | `.bg-stars` | 多个 `radial-gradient` 模拟星空粒子，配合 `starDrift` 动画缓慢漂移 |
| 2 | `.bg-nebula` | 三个 `radial-gradient` 椭圆叠加，模拟紫/绿/蓝星云光晕 |
| 3 | `.bg-grid` | 网格线（`linear-gradient`），用 `mask-image` 从中心向外渐隐 |
| 4 | `.bg-scanlines` | 重复水平线条，模拟 CRT 扫描线效果 |

### 2.2 配色与字体

- **CSS 变量**：沿用欢迎页的 `--sol-purple`、`--sol-green`、`--sol-blue`、`--phantom-pink`、`--warp-magenta`、`--bn-yellow`，并新增语义化变量（`--bg-card`、`--border-subtle`、`--text-primary` 等）。
- **字体**：通过 Google Fonts 引入 Orbitron（标题/表头/排名）+ Exo 2（正文/按钮/标签），与欢迎页一致。
- **涨跌色**：正面 `#14F195`（Solana 绿），负面 `#ff4d6a`（亮红），替代原来的 `#22c55e` / `#ef4444`。

### 2.3 页面标题

- 使用 Orbitron 字体，`font-weight: 700`。
- 三色渐变（紫→蓝→绿），配合 `titleShift` 动画持续流动。
- 前缀 `⟡` 符号呼应欢迎页的品牌风格。

### 2.4 Glassmorphism 卡片

表格容器 `.table-card`：
- `background: rgba(15, 12, 30, 0.65)` 半透明暗色。
- `backdrop-filter: blur(16px)` 毛玻璃效果。
- `border: 1px solid rgba(153, 69, 255, 0.12)` 微光边框。
- `border-radius: 16px` 圆角。
- `::before` 伪元素在顶部生成紫→蓝渐变发光线。

调度器状态栏 `.scheduler-bar` 也采用同样的毛玻璃风格。

### 2.5 排名徽章

前三名使用差异化样式：

| 排名 | 类名 | 颜色 | 效果 |
|------|------|------|------|
| #1 | `.rank.gold` | `--bn-yellow` 金色 | 渐变背景 + 金色边框 + 发光阴影 |
| #2 | `.rank.silver` | `#c0c0d2` 银色 | 渐变背景 + 银色边框 |
| #3 | `.rank.bronze` | `#cd7f32` 铜色 | 渐变背景 + 铜色边框 |
| 4+ | `.rank` | 暗紫色 | 统一低调样式 |

徽章为 `28×28px` 圆角方块，使用 Orbitron 字体。

### 2.6 Tab 导航

- 从圆角按钮改为**下划线指示器**风格。
- 激活态显示 `border-bottom: 2px solid var(--sol-purple)` + 紫色文字 + 文字发光。
- 底部有统一的分割线（`::after` 伪元素）。

### 2.7 操作按钮

更新按钮采用渐变背景：
- 默认：`linear-gradient(135deg, rgba(153,69,255,0.3), rgba(0,209,255,0.2))` + 紫色边框。
- Hover：`::before` 伪元素覆盖渐变层 + 发光阴影 + 上移 1px。
- 按钮文字包裹在 `<span>` 中（`z-index: 1`），确保渐变层不遮挡文字。

### 2.8 表格细节

- **表头**：Orbitron 字体，大写字母，`letter-spacing: 0.08em`，粘性定位（`position: sticky`）。
- **代币列**：Flex 布局，logo 图片 30px 圆形 + 代币名称（文字溢出截断）。
- **符号列**：`--sol-blue` 青色高亮，加粗。
- **LP 状态标签**：彩色 tag 组件（绿色 = 已burn/锁，红色 = 否，灰色 = 未知）。
- **持币地址**：添加 `toLocaleString()` 千分位格式化。
- **行悬浮**：背景色微亮 + 分割线颜色加深。

### 2.9 描述区域

从纯文本改为带左侧紫色边线的卡片样式（`border-left: 2px solid`），淡紫色背景。

### 2.10 其他增强

- 自定义滚动条：6px 宽，紫色半透明滑块。
- 加载动画：文字后带旋转的紫色圆环 spinner。
- `formatCompact()` 新增 B（十亿）级别格式化。
- 移动端响应式：768px 以下表格可水平滚动，间距缩小。
