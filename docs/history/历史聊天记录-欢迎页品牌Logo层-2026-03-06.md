## 1. 需求：背景 logo 特效加入欢迎页

**用户需求**  
- 将 `背景logo.html` 中的「品牌 Logo 层」特效加入欢迎页面，其他不改动。  
- 品牌 Logo 层包含：  
  - **Solana ×3**：经典三条斜杠 logo，紫绿渐变，左上 / 右下 / 中上，大小不同形成纵深。  
  - **Polymarket ×3**：P 字母 + 圆环，冰蓝色调，带虚线轨道装饰。  
  - **Binance ×3**：菱形五钻结构，金黄渐变。  
- 每个 logo 需具备：  
  - 三组独立幻影浮现动画（phantomFloat1/2/3）：不同节奏的显隐、缩放、位移、模糊。  
  - 持续旋转（正转/反转，25–45s 不同周期）。  
  - `mix-blend-mode: screen` 融入暗色背景。  
  - 光晕脉冲（`::before` 伪元素 radial glow）。  
  - 鼠标视差：随鼠标位置轻微偏移（用 `transform`，不用 margin）。

**实现（背景logo.html）**  
- 在 `背景logo.html` 中升级既有 `brand-logos` 区块：  
  - 每个 logo 拆成 4 层：3 个幻影层（phantom-1/2/3）+ 1 个核心层（core），分别套用 phantomFloat1/2/3 与不同 delay/duration。  
  - 使用 SVG `<symbol>` + `<use>` 复用图形，在隐藏的 `brand-logo-defs` 中定义渐变与 symbol（mark-solana / mark-poly / mark-bn）。  
  - 旋转动画保留在 `.brand-logo` 上（phantomSpin1 / phantomSpinReverse），周期 29–45s。  
  - 视差由 JS 设置 CSS 变量 `--px`、`--py`，容器用 `transform: translate3d(var(--px), var(--py), 0)`。

## 2. 需求：实际欢迎页也要有这些 logo

**用户反馈**  
- 「增加的币安 logo、polymarket logo、solana logo 呢」——此前只改动了 `背景logo.html`，实际对外服务的是 `src/public/index.html`，欢迎页上没有这些 logo。

**实现（src/public/index.html）**  
- 在 `src/public/index.html` 中完整同步品牌 Logo 层：  
  1. **:root** 增加 `--bn-yellow: #F0B90B`。  
  2. **CSS**：在 distortion-orb 与 phantom-ring 之间插入整段「FLOATING BRAND LOGOS」样式（.brand-logos、.brand-logo-defs、.brand-logo、.logo-layer、.core/.phantom、.logo-sol-* / .logo-poly-* / .logo-bn-*、phantomFloat1/2/3、phantomSpin1/Reverse、logoGlowPulse、::before 光晕）。  
  3. **HTML**：在 `light-streak-2` 与 `glyphs` 之间插入 `brand-logos` 容器，内含隐藏的 SVG defs（solGrad / polyGrad / bnGrad，以及 mark-solana / mark-poly / mark-bn），以及 9 个 `.brand-logo`（sol-1/2/3、poly-1/2/3、bn-1/2/3），每个含 4 个 `.logo-layer`（3×phantom + 1×core）。  
  4. **JS**：在现有 mousemove 视差逻辑中增加对 `.brand-logo` 的循环，用 `setProperty('--px', ...)` 与 `setProperty('--py', ...)` 驱动位移。
